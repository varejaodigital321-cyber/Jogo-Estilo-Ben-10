# Spec Kit

Este diretório é a raiz de governança Spec Kit do projeto. O fluxo obrigatório
é:

`Constituição → Especificação → Esclarecimento → Plano técnico → Tarefas → Análise de consistência → Implementação → Convergência`

O Spec Kit define o que é autorizado e quando uma etapa pode avançar. O
obra/superpowers define como o agente executa a etapa autorizada, com seleção
automática da skill pertinente. Nenhuma camada substitui a outra.

As fontes normativas são `AGENTS.md`, `memory/constitution.md` e
`skill-policy.yaml`. A referência de raiz está em
`../SUPERPOWERS_GOVERNANCE.md`. A integração Codex do Spec Kit e suas skills
estão instaladas neste repositório.
