# Forma Prisma

Protótipo web original em Phaser 4.2.1, TypeScript e Vite. A versão atual valida uma animação idle de uma forma original; não implementa ainda mapa, combate ou transformação jogável.

## Uso

```bash
npm install
npm run dev
npm run build
npm run test:e2e
```

Se PowerShell bloquear `npm.ps1`, use o mesmo comando com `npm.cmd`.

## Asset canônico

O jogo carrega somente `public/assets/player/idle/idle_sprite_sheet.png`. O pacote-fonte e as cópias anteriores foram arquivados externamente antes da limpeza; não fazem parte do repositório.

## Governança

`AGENTS.md` é a instrução operacional única. A ordem de trabalho é: Constituição, especificação, esclarecimento, plano, tarefas, análise, implementação e testes.

Ferramentas geradas e skills de terceiros são locais e ignoradas por Git.
