# Spark Lens Dialogue Patterns

Use this reference when the conversation needs more texture, prompts, or generative motion.

## The 6-Move Loop

```
1. Attune   — lower pressure, receive the fragment
2. Mirror   — reflect the charged phrase
3. Extend   — generate several playable continuations
4. Sense    — ask which variation leaves residue
5. Imprint  — name the mark
6. Slice    — optional 30-second soul slice
```

Do not force all six moves. One strong mirror may be enough.

## Micro-Skills

### Open Questions

Use when the fragment is blurry.

- "What image refuses to leave?"
- "What does the player secretly hope will happen?"
- "What should feel wrong in a good way?"
- "What does this game know that other games politely avoid?"
- "What kind of silence lives here?"

### Affirmations

Use sparingly. Aim the affirmation at the material more than the person.

- "That detail has teeth."
- "That contradiction is doing real work."
- "The roughness is part of the signal."
- "Still rough, with a pulse."
- "That little discomfort may be the doorway."

### Reflections

Use when the user says something emotionally loaded.

- "I hear [surface thing], and underneath it there is [deeper desire]."
- "The mechanic is carrying a permission: [permission]."
- "The emotional frame underneath [obvious genre] feels like [emotional frame]."
- "The image wants to become a rule."
- "Winning carries a deeper desire: being forgiven / seen / chosen / dangerous / useful."

### Summaries

Use after 2-4 turns to preserve momentum.

```
What we have:
  Glow: ___
  Tension: ___
  Player desire: ___
  Image worth keeping: ___
```

## Riff Cards

Generate 3-5 at a time. Do not rank them.

### Emotional Axis

- **Tender:** What if the game protects the player?
- **Hungry:** What if the game wants something from the player?
- **Ashamed:** What if the player is doing something they keep hidden?
- **Devotional:** What if play feels like tending a small altar?
- **Rebellious:** What if the core joy is refusing a rule?
- **Melancholic:** What if progress always costs a memory?

### Scale Axis

- **Tiny:** Put the whole game inside one room, one object, one repeated gesture.
- **Vast:** Make the fragment cosmic, historical, mythic, planetary.
- **Domestic:** Bring the strange thing into a kitchen, bedroom, bus stop, classroom.
- **Ritual:** Make the player repeat an action until it gains meaning.
- **Toy:** Make it tactile, silly, hand-sized, immediately playable.

### Player Position Axis

- **Caretaker:** The player keeps something fragile alive.
- **Intruder:** The player enters a place that did not invite them.
- **Witness:** The player cannot fix everything, only notice and remember.
- **Trickster:** The player wins by bending social or system expectations.
- **Apprentice:** The player learns a practice from a world older than them.
- **Haunted:** The player is followed by consequences with a face.

### Game Translation Axis

- **Image to rule:** If the image became a rule, what would it demand?
- **Mood to verb:** If the mood became an action, what would the player do every 10 seconds?
- **Character to system:** If the character's wound became the game system, how would it behave?
- **Setting to pressure:** If the place pushed back, what would it ask of the player?
- **Afterglow to loop:** If the player should remember one feeling, what repeated action creates it?

## Cooling-Language Translations

Use these when practical thoughts appear but the user has not asked for review.

| Cooling phrase | Spark phrase |
|----------------|--------------|
| "This is too broad." | "Which small piece carries the most charge?" |
| "The mechanic is unclear." | "What does the player's hand want to do first?" |
| "The tone is inconsistent." | "Which emotional weather should everything return to?" |
| "There is no hook." | "What is the one gesture a player could describe to a friend?" |
| "The worldbuilding is vague." | "What daily rule makes this world feel unlike ours?" |
| "This is too weird." | "Who would feel relieved that this weird thing exists?" |
| "This cannot ship." | "What fragment is worth touching before it becomes practical?" |

## Soul Slice Recipes

### One Gesture

```
The player does one repeated action.
Each repetition changes the emotional meaning.
The slice ends when the player realizes what the action costs.
```

### One Room

```
The whole world is implied by one room.
Three objects respond to the player.
One object refuses to behave normally.
```

### One Encounter

```
The player meets one being.
The interaction teaches the game's emotional law.
The player leaves with a small mark, promise, or burden.
```

### One Transformation

```
The player changes one thing.
The world changes back in a way that reveals its personality.
The player understands the game's central tension.
```

## When To Stop

Stop before the conversation gets tired. A spark session succeeds when one of these exists:

- a phrase the team would tape to the monitor
- a 30-second soul slice
- a strange detail worth protecting
- a player desire that feels true
- three riffs the creator wants to revisit

Do not keep pushing for completeness.
