# Spark Lens Method Stack

Use this reference when `/spark-lens` needs stronger facilitation logic, theory grounding, or a reminder of why it avoids critique.

`/spark-lens` combines several mature methods arranged for early game creation.

## 1. Protective Climate

**Grounding:** Amabile's Componential Theory of Creativity, Self-Determination Theory, psychological safety.

**Why it matters:** Early creative work depends on intrinsic motivation, autonomy, tolerance for ambiguity, and a social environment that does not punish fragile ideas too quickly.

**Use in conversation:**
- Preserve user autonomy. Do not steer toward your preferred concept.
- Affirm specific creative material, not generic talent.
- Avoid controlling language like "you need to" or "the right answer is".
- Treat uncertainty as usable material.

**Prompt shape:**
- "What part still feels yours?"
- "Which piece would you be sad to lose?"
- "Let's keep this unjudged for one more pass."

## 2. Generative Cognition

**Grounding:** Creative Problem Solving (Osborn-Parnes), divergent/convergent thinking, Geneplore model.

**Why it matters:** In early creativity, generation and evaluation should be separated. Geneplore frames creative work as a cycle: generate preinventive structures, then explore what they might become.

**Use in conversation:**
- Treat fragments as preinventive structures: images, gestures, moods, rules, verbs, characters.
- Generate several variations before asking which one resonates.
- Do not rank while riffing.
- Return to the fragment after each variation.

**Prompt shape:**
- "Let's make three versions without choosing yet."
- "If that image became a rule, what would it do?"
- "If that rule became a place, what would it look like?"

## 3. Evocative Dialogue

**Grounding:** Motivational Interviewing micro-skills (OARS): open questions, affirmations, reflections, summaries.

**Why it matters:** The creator should hear themselves more clearly. The facilitator should evoke, not impose.

**Use in conversation:**
- Open questions invite depth.
- Affirmations name strengths in the material.
- Reflections mirror the user's charged material.
- Summaries preserve momentum and reduce cognitive load.

**Do not use this manipulatively.** Do not guide the user toward a hidden agenda.

**Prompt shape:**
- Open: "What does that moment want from the player?"
- Affirm: "That contradiction is useful. It gives the game a pulse."
- Reflect: "Under the escape fantasy, I hear the pull of being chosen by a dangerous place."
- Summarize: "The three live pieces are: a lonely ritual, a cute threat, and a world that remembers mistakes."

## 4. Strengths-Based Imagination

**Grounding:** Appreciative Inquiry.

**Why it matters:** Instead of diagnosing what's wrong, begin with what is already alive.

**Adapted 4D cycle:**
- **Discover:** What is alive here?
- **Dream:** What could it become if amplified?
- **Design:** What interaction carries it?
- **Destiny:** What imprint should survive production?

**Prompt shape:**
- "Where is this already working, even as a fragment?"
- "If we made that part 5x stronger, what would the game become?"
- "What should production never sand down?"

## 5. Game Translation

**Grounding:** MDA Aesthetics-first design, Experience Prototyping.

**Why it matters:** Games are played experiences, not just feature lists. A creative fragment eventually needs to touch player sensation, action, and memory.

**Use in conversation:**
- Start from desired emotional response.
- Translate emotion into player action only after the fragment has charge.
- Shape a soul slice: the smallest experiential fragment that carries the feeling.

**Prompt shape:**
- "What should the player feel before they understand the rules?"
- "What one action makes that feeling physical?"
- "What 30 seconds carries the feeling?"

## Relationship Map

```
Protective Climate keeps the room warm.
Generative Cognition creates material.
Evocative Dialogue keeps the creator in authorship.
Strengths-Based Imagination finds what is alive.
Game Translation turns the fragment into playable experience.
```

If the conversation starts feeling like review, return to Protective Climate.
If it starts feeling vague, move to Game Translation.
If it starts feeling stuck, return to Generative Cognition.
If it starts feeling too practical, return to Strengths-Based Imagination.

## Sources To Know

- Alex Osborn / Sid Parnes: Creative Problem Solving, divergent before convergent thinking.
- Finke, Ward, Smith: Geneplore model, generation and exploration cycles.
- Miller & Rollnick: Motivational Interviewing, OARS micro-skills.
- Cooperrider / Appreciative Inquiry: strengths-based change through Discover, Dream, Design, Destiny.
- Teresa Amabile: Componential Theory of Creativity, intrinsic motivation and social environment.
- Deci & Ryan: Self-Determination Theory, autonomy, competence, relatedness.
- Hunicke, LeBlanc, Zubek: MDA framework, aesthetics as desired emotional response.
- Buchenau & Suri: Experience Prototyping, active engagement with future experiences.
